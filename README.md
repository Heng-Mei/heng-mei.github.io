# Heng-Mei Blog

这是我的个人技术博客源码仓库，使用 **Hugo + FixIt** 构建，  
主要记录算法、学习笔记以及一些技术随想。

🌐 博客地址：<https://heng-mei.github.io/>

## 技术栈

- **Static Site Generator**: Hugo
- **Theme**: FixIt
- **Hosting**: GitHub Pages
- **CI/CD**: GitHub Actions

## 目录结构

```text
content/
├── algorithms/   # 算法与题解
├── notes/        # 学习笔记
└── posts/        # 随想

static/
└── favicon.ico           # 站点图标
└── avatar.jpg            # 个人头像
└── ...                   # 其他资源文件

themes/FixIt/             # Hugo 主题
```

- 本仓库用于博客源码管理与个人记录
